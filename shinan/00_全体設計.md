# 競艇予想AI研究所

これは競艇予想AI「フナヨミAI」を動かす前に読む研究所の中核ファイル。
詳細は各ファイルに分けるが、この1枚で思想・順番・禁止事項・現在の学びを確認できるようにする。

## 一言でいうと

```text
当てるAIではなく、買う価値があるレースだけを残すAI。
```

的中率だけを追うと、安い本命を拾って回収率が死ぬ。
回収率だけを追うと、穴狙いで再現性が死ぬ。

フナヨミAIはその中間で、公式データから「買っていい条件」を段階的に絞る。

## 必ず守ること

1. モーターデータは毎回、各会場の公式 `rankingmotor` を使う。
2. 過去検証では、結果・払戻・最終人気・締切後オッズを予想入力に混ぜない。
3. オッズは本番の現在オッズだけ、最終購入判断に使う。
4. アレドで落ちたレースは、高度分析AIに進ませない。
5. `HUMAN_CHECK` は基本的に買わない。自動運用では見送り寄りに扱う。
6. 買わない判断も予想の一部としてログに残す。
7. 勝った日だけで判断しない。負けログを消さない。

## 分析フロー

```text
1. 出走表を読む
2. 公式モーターデータを取得する
3. 直前情報を読む
4. モーター番長AIで機力を相対評価する
5. 展示・進入チェックAIで直前気配を評価する
6. 荒れ度診断AI / アレドで参加可否を決める
7. 通過レースだけ高度分析AIで買い目を出す
8. 本番だけオッズ番人AIで安すぎる券を削る
9. 結果を保存し、検証ログへ戻す
```

## 各AIの役割

### モーター番長AI

モーターだけを見る。
公式モーター順位、2連率、3連率、前検タイム、相対順位を見て、各艇を `◎ / ○ / △ / ×` で評価する。

モーター番長は買い目を決めない。
アレドへ「1号艇が弱い」「外枠に強モーターが複数いる」などの危険材料を渡す。

詳細: [スキル/01_モーター番長AI.md](スキル/01_モーター番長AI.md)

### 展示・進入チェックAI

展示タイム、スタート展示、進入ズレ、チルト、部品交換、風、波を読む。

現状まだ弱い部分。
特に `1-2-5`、`1-2-6`、2コース差し、外伸びを拾うために強化する。

詳細: [スキル/02_展示進入チェックAI.md](スキル/02_展示進入チェックAI.md)

### 荒れ度診断AI / アレド

買う前のゲート。
星が少ない、根拠が薄い、モーターだけで荒れ判定しているレースを落とす。

アレドで落ちたら終了。
ここを通過したレースだけが高度分析AIへ行ける。

詳細: [スキル/03_荒れ度診断AI_アレド.md](スキル/03_荒れ度診断AI_アレド.md)

### 高度分析AI

通過レースだけ買い目を作る。
基本は1号艇軸だが、1号艇が弱い時に無理やり買わない。

現在の重要ルールは `1-2-5 / 1-2-6` の保険。
2号艇が強い時、または2コース差し気配がある時は軽視しない。

詳細: [スキル/04_高度分析AI.md](スキル/04_高度分析AI.md)

### オッズ番人AI

本番専用。
予想AIが出した買い目候補を、現在オッズで買う価値があるか見る。

オッズ番人AIは新しい買い目を作らない。
安すぎる券を削る、点数を減らす、見送る判断をする。

詳細: [スキル/05_オッズ番人AI.md](スキル/05_オッズ番人AI.md)

## 現在の検証結果

- 2026-06-20 ランダム16R: 通過10R、的中2、回収率192.2%
- 2026-06-26 鳴門1R〜10R 旧ルール: 通過8R、的中0、回収率0.0%
- 2026-06-26 鳴門1R〜10R 新ルール再集計: 通過4R、的中3、回収率372.9%
- 2026-07-01 ランダム30R: 通過16R、的中4、回収率244.4%
- 2026-05 ランダム30R: 通過11R、的中3、回収率37.1%
- 2026-05 ランダム100R 現行軽量モデル: 購入21R、的中2、回収率26.8%

結論:

```text
弱くはない。
でも、まだ伝説の予想師ではない。
一撃依存があり、100Rでは安定していない。
```

詳細: [ログ/検証ログ.md](ログ/検証ログ.md)

## 現在の改善テーマ

1. 展示・進入チェックAIを本格実装する。
2. オッズ番人AIで安い本命買いを削る。
3. 会場別補正を入れる。
4. 星5レースの点数を減らす。
5. 星4レースは、モーター以外の追加根拠がある時だけ買う。
6. 星4を購入星4と穴観測星4に分ける。

詳細: [ログ/改善仮説.md](ログ/改善仮説.md)
