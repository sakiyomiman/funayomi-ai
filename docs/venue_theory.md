# おすすめ会場の理論 — なぜ「場を選ぶ」のか

サキヨミAIの一番大きな発見は、買い方の工夫よりも「**どの場で勝負するか**」の方がはるかに結果を左右する、ということでした。
このドキュメントは、677レースの実測データからその理由を説明します。

---

## 1. 同じAIでも、場によって結果は倍近く違う

同じエンジン・同じ買い方で検証した結果です（2026年6〜7月）。

| 会場 | 検証数 | 回収率 | 評価 |
|---|---|---|---|
| 尼崎 | 81レース | 115% | 勝てる場 |
| 浜名湖 | 53レース | 112% | 勝てる場 |
| 蒲郡 | 110レース | 106% | 勝てる場 |
| 福岡 | 60レース | 99% | 互角 |
| 桐生 | 28レース | 85% | 苦手 |
| 津 | 25レース | 68% | 苦手 |
| 住之江 | 31レース | 67% | 苦手 |

ロジックは1文字も変えていません。**変えたのは場所だけ**です。

## 2. 「イン最強の場」ほど勝てない理由

意外に思われるかもしれませんが、住之江（1コース1着率61.8%・全国屈指のイン水面）は最下位クラスでした。

理由はオッズにあります。「住之江のインは堅い」は誰でも知っている情報なので、
1号艇がらみの舟券は徹底的に買われていて、**当たっても配当が安すぎる**のです。
競艇は売上の約25%が控除されるため、「みんなが正解を知っているレース」で勝ち続けることは構造的にできません。

**予想が当たることと、お金が増えることは、別の問題**です。ここを混ぜると負けます。

## 3. 勝てる場の共通点

勝てた3場（蒲郡・尼崎・浜名湖）には共通の性質がありました。

- **インがそこそこ負ける**（1号艇の1着率が50%前後。実測では蒲郡は49%でした）
- **ただし、来る相手は読みやすい**

蒲郡の実測データ（55レース）が分かりやすい例です。

- 1号艇が1着のとき、2着に来るのは **2号・3号・4号がそれぞれ約30%**、きれいに三等分
- 一方、**5号・6号が2着に来たケースは0%**

つまり「1号アタマ、相手は2・3・4号」という素直なフォーメーションが、この場では統計的に正しい。
インが半分負けるおかげでオッズにうまみが残り、かつ相手選びで迷わない——これが「AIと相性のいい水面」の正体です。

## 4. 「参加しない」が一番の武器

サキヨミAIの荒れ度AIは、検証期間中の全レースのうち**約半分を「見送り」と判定**しています。
見送り判定の的中率（＝実際に荒れた、または1号艇が飛んだ割合）は約7割で、
34,790円の大波乱レースを事前に回避した例もあります。

全レースに参加すると回収率84%。参加レースを選ぶと110%。
**この差は「どう買うか」ではなく「どこで買わないか」から生まれています。**

## 5. この理論の限界も書いておきます

- 検証は2026年6〜7月の677レースです。**季節・節・選手層が変われば数字は動きます。**
- 「勝てる場」も未来永劫勝てる保証はありません。だからサキヨミAIは毎晩学習を回して、場の評価を更新し続ける設計にしています。
- サンプルの少ない場（検証30レース未満）の数字は、まだ偶然の範囲を出ていません。

数字は生き物です。このドキュメントの内容も、データが増えれば書き換わります。
それがこのプロジェクトの正しい姿だと考えています。
